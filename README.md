# PDA-Project

Language L = {an bn | n >= 0} <br>
generated by context free grammar G = ({a,b,S}, {a,b}, R, S)
  where R contains rules S->aSb and S->e <br>
The resulting PDA is <br>
  M1 = ({p,q,qa,qb,q$}, {a,b}, {a,b,S}, rules, p, {q$}) <br>
  where rules = <br>
      1 (p,e,e)  -> (q,S) <br>
      2 (q,a,e)  -> (qa,e) <br>
      3 (qa,e,a) -> (q,e) <br>
      4 (q,b,e)  -> (qb,e) <br>
      5 (qb,e,b) -> (q,e) <br>
      6 (q,$,e)  -> (q$,e) <br>
      7 (qa,e,S) -> (qa,aSb) <br>
      8 (qb,e,S) -> (qb,e) <br>
